# HubCredo Authentication system

## Setup Instructions:

- Install all the dependencies:
  ```bash
  npm install
  ```
- Create a Supabase project and copy Supabase URL and Key to ENV file:
  ```bash
  VITE_SUPABASE_URL=
  VITE_SUPABASE_PUBLISHABLE_KEY=
  ```
- The project is ready to run.
  ```bash
  npm run dev
  ```

  ## Project Tech Stack:

  1. **Frontend Library**: React.js with TypeScript.
  2. **Component Library**: ShadCN UI Components.
  3. **Authentication**: Supabase authentication.
  4. **Database**: PostgreSQL.
